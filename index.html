JavaScript

JavaScript用於處理網頁的互動，例如:Event, Function......

請記得分檔案邊寫腳本

變數

  • var

    • 儲存可變動的數值、文字，屬於函式作用域。

    若應用於if、else、while、for的話，會發生此變數從區域變數變成全域變數的情況。

  • let

    • 儲存可變動的數值、文字，作用域為其所包含的{}。

    於全域呼叫的變數與函式內呼叫的變數不會起衝突。

  • const

    • 儲存不可變動的數值、文字，需要在宣告時設定常數。

方法(Method)

  • querySlector
  • getElementById

函式(Function)

  • alert

    • alert('字串')

    將字串以彈跳視窗的方式呈現於網頁上方。

  • prompt

    • prompt('字串')

    與alert函式同理，但多了一條輸入框格，拿到輸入資料。

事件(Event)

addEventListener事件 V.S. onclick事件

錯誤寫法：obj.onclick= fn(param)或obj.onclick= fn(): 因為這樣寫函式會立即執行，不會等待點選觸發，特別注意一下
